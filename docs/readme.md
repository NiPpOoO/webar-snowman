# WEBAR-SNOWMAN

Демо WebAR со снеговиком на NFT-маркере (A‑Frame 1.3.0 + AR.js).

## Структура
- assets/
  - markers/snowman/
    - snowman.fset
    - snowman.fset3
    - snowman.iset
  - models/
    - snowman.glb
    - light.glb
- src/
  - script.js
  - style.css
- index.html

## Локальный запуск
```bash
npm i
npm run start
# открой http://localhost:8080
